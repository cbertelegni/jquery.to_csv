jquery.to_csv
=============

## Plugin Jquery para descargar tabla html en formato csv  

### Ejemplo de uso:
Incluir en el head el archivo "jquery.to_csv.min.js"
```
<script type="text/javascript" src="jquery.to_csv.min.js"> </script>
```
Luego descargar los datos... 
```
$("table").to_csv();

```
